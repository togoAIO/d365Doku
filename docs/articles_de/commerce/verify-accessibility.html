    Überprüfen der Zugänglichkeit des Seiteninhalts

    Important

    Dynamics 365 Retail ist jetzt Dynamics 365 Commerce und bietet umfassende Handelsfunktionen für alle Kanäle – von E-Commerce über Shops bis hin zu Callcentern. Weitere Informationen zu diesen Änderungen finden Sie unter Microsoft Dynamics 365 Commerce.

    In diesem Thema wird beschrieben, wie Sie die Barrierefreiheit von Seiteninhalten in Microsoft Dynamics 365 Commerce überprüfen.

    Übersicht

    Wenn Sie mit dem Ändern einer Seite fertig sind, sollten Sie sicherstellen, dass der Inhalt für alle im Web zugänglich ist. In den Commerce-Authoring-Tools können Sie mithilfe des integrierten Dienstes Microsoft Accessibility Insights auf einfache Weise die Zugänglichkeit von Seiteninhalten überprüfen. Dieser Dienst überprüft Ihren Seiteninhalt anhand der neuesten Richtlinien zur World Wide Web Consortium (W3C Zugänglichkeit).

    Die Microsoft Accessibility Insights-Integration muss auf Mandanten- oder Site-Ebene aktiviert sein, bevor Sie sie verwenden können.

    Aktivieren Sie Microsoft Accessibility Insights für alle Websites in Ihrem Mandanten

    Um Microsoft Accessibility Insights zu aktivieren, führen Sie die folgenden Schritte aus, um alle Commerce-Sites in Ihrem Mandanten zu integrieren.

    Note

    Um auf Mandanteneinstellungen zuzugreifen, müssen Sie als Systemadministrator bei Commerce angemeldet sein.

    1. Melden Sie sich bei Commerce als Systemadministrator an.
    2. Wählen Sie im linken Navigationsbereich Mandanteneinstellungen (neben dem Zahnradsymbol) aus, um es zu erweitern.
    3. Wählen Sie unter Mandanteneinstellungen die Option Funktionen aus.
    4. Legen Sie die Option Zugänglichkeitsprüfung auf Ein fest.

    Aktivieren Sie Microsoft Accessibility Insights für eine einzelne Site

    Um die Microsoft Accessibility Insights-Integration für eine einzelne Commerce-Site zu aktivieren, führen Sie die folgenden Schritte aus.

    1. Wählen Sie unter Sites Fabrikam (oder den Namen Ihrer Site) aus.
    2. Wählen Sie im linken Navigationsbereich Site-Einstellungen aus, um den Bereich zu erweitern.
    3. Wählen Sie unter Site-Einstellungen die Option Funktionen aus.
    4. Legen Sie die Option Zugänglichkeitsprüfung auf Ein fest.

    Überprüfen Sie die Zugänglichkeit des Inhalts auf der Homepage

    Um den integrierten Microsoft Accessibility Insights-Dienst zu verwenden und so den Inhalt Ihrer Startseite in Commerce zu scannen und zu prüfen, gehen Sie folgendermaßen vor.

    1. Wählen Sie unter Sites Fabrikam (oder den Namen Ihrer Site) aus.
    2. Wählen Sie im linken Navigationsbereich die Option Seiten aus.
    3. Suchen Sie die Startseite und wählen Sie sie aus, um sie im Seiteneditor zu öffnen.
    4. Wählen Sie in der Befehlsleiste die Option Zugänglichkeit prüfen aus. Die Seite Zugänglichkeit prüfen wird angezeigt.
    5. Überprüfen Sie nach Abschluss des Scans den Inhalt des Berichts.
    6. Wenn Prüfungen fehlgeschlagen sind, wählen Sie jedes fehlgeschlagene Prüfelement aus, um es zu erweitern und weitere Details anzuzeigen.
    7. Um den Bericht zu schließen, nachdem Sie ihn geprüft haben, scrollen Sie zur Seite Zugänglichkeit prüfen, und wählen Sie OK aus.
