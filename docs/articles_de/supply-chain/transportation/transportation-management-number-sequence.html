    Transportverwaltung Sequenznummer

    Important

    Dynamics 365 for Finance and Operations hat sich zu speziell entwickelten Anwendungen entwickelt, mit denen Sie bestimmte Geschäftsfunktionen verwalten können. Weitere Informationen zu diesen Änderungen finden Sie im Dynamics 365-Lizenzierungshandbuch.

    Verwenden Sie die Seite Nummernfolgen im Modul Transportverwaltung, um verschiedene Pro-Nummern festzulegen. Pro-Nummern werden von Spediteuren verwendet, um den Fortschritt der einzelnen Sendungen zu organisieren und zu verfolgen.

    Erstellen einer Nummernfolge für eine Pro-Nummer

    Um eine Sequenz für eine Pro-Nummer zu erstellen, gehen Sie wie folgt vor:

    1. Gehen Sie zu Transportverwaltung > Einrichten > Spediteure > Nummernfolgen.
    2. Wählen Sie Neu, um eine neue Sequenz für eine Nummer zu erstellen.
    3. Geben Sie eine eindeutige ID und einen beschreibenden Namen für die Nummernfolge ein.
    4. Im Feld Nummernfolge-Typ ist Pro-Nummer die einzige Option.
    5. Im Feld Prüfziffer ist Prüfziffer die einzige Option und ist als generische Engine festgelegt.
    6. Geben Sie auf dem Inforegister Sequenz Informationen über die Sequenz an.
    7. Schließen Sie die Seite.

    Verknüpfen einer Sequenz mit einem Spediteur

    Um eine Sequenz mit einem Spediteur zu verknüpfen, gehen Sie wie folgt vor:

    1. Wechseln Sie zu Transportverwaltung > Einstellungen > Spediteure > Spediteure.
    2. Wählen Sie einen Spediteur aus.
    3. Wählen Sie Bearbeiten aus.
    4. Wählen Sie auf der Inforegister-Registerkarte Übersicht eine Option im Feld Pro Nummernfolge.
    5. Schließen Sie die Seite.
