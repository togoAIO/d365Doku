    Ändern einer Servicestückliste

    Important

    Dynamics 365 for Finance and Operations hat sich zu speziell entwickelten Anwendungen entwickelt, mit denen Sie bestimmte Geschäftsfunktionen verwalten können. Weitere Informationen zu diesen Änderungen finden Sie im Dynamics 365-Lizenzierungshandbuch.

    Sie haben die Möglichkeit zum Erfassen der Historie eines Elements innerhalb einer Servicestückliste. Jedes Mal, wenn Sie eine Stücklistenposition aktualisieren, wird eine Historieposition im Bereich Verlauf erstellt. Die Historieposition zeigt den aktuellen Status der Stücklistenposition an.

    Aktualisieren eines Servicestücklistenelements

    1. Klicken Sie auf den Bereichsseitenknoten: Serviceverwaltung > Gemeinsam > Servicevereinbarungen > Servicevereinbarungen.

    2. Klicken Sie auf Bearbeiten, um das Formular Serviceverträge zu öffnen.

    3. Klicken Sie im Aktivitätsbereich auf Serviceobjekte, um das Formular Serviceobjekte zu öffnen.

    4. Wählen Sie das Objekt aus, für das Sie eine Stücklistenposition aktualisieren möchten, und klicken Sie anschließend auf Designer.

    5. Wählen Sie im Formular Designer die zu aktualisierende Stücklistenposition aus, und klicken Sie anschließend auf Stücklistenposition bearbeiten.

      Note

      Aktivieren Sie auf der Registerkarte Einrichtung das Kontrollkästchen Beim Hinzufügen bearbeiten, wenn das Formular Stücklistenposition bearbeiten geöffnet werden soll, wenn Sie eine Position in die Servicestückliste ziehen.

    6. Geben Sie im Feld Menge die Menge ein.

    7. Aktivieren Sie das Kontrollkästchen Serviceauftragsposition erstellen, wenn für den Ersetzungsartikel eine (fakturierbare) Serviceauftragsposition erstellt werden soll.

    8. Klicken Sie auf OK, um das Formular zu schließen.

    Löschen einer Servicestücklistenposition

    1. Klicken Sie auf den Bereichsseitenknoten: Serviceverwaltung > Gemeinsam > Servicevereinbarungen > Servicevereinbarungen.

    2. Klicken Sie auf Bearbeiten, um das Formular Serviceverträge zu öffnen.

    3. Klicken Sie im Aktivitätsbereich auf Serviceobjekte, um das Formular Serviceobjekte zu öffnen.

    4. Wählen Sie das Objekt aus, aus dem eine Servicestücklistenposition gelöscht werden soll, und klicken Sie anschließend auf Designer.

    5. Wählen Sie im Formular Designer die zu Stücklistenposition aus, und klicken Sie anschließend auf Stücklistenposition löschen.
