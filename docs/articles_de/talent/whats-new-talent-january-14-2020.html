Neuerungen oder Änderungen in Dynamics 365 Talent (14. Januar 2020)

Important

Dynamics 365 Talent: Attract- und Onboard-Apps werden eingestellt. Weitere Informationen finden Sie unter Einstellen von Dynamics 365 Talent: Attract- und Onboard-Apps.

[!include[rename-banner](~/includes/cc-data-platform-banner.md)]

Dieser Artikel beschreibt Funktionen, die entweder neu oder geändert in Dynamics 365 Talent sind.

Änderungen in Attract

Diese Version enthält kleinere Fehlerkorrekturen für Dynamics 365 Talent: Attract.

Änderungen in Onboard

Diese Version enthält kleinere Fehlerkorrekturen für Dynamics 365 Talent: Onboard.

Core HR-Änderungen

Die in diesem Abschnitt beschriebenen Änderungen gelten für Buildnummer 8.1.2709. Die Zahlen in Klammern in einigen Überschriften beziehen sich auf Supportnummern in Microsoft Dynamics Lifecycle Services (LCS).

System kann beim Import über Data Management Framework keine Kalendertage generieren (381195)

Diese Änderung behebt ein Problem, bei dem beim Importieren von Kalendertagen der Fehler Ungültiges Datumsformat auftritt. Dieser Fehler verhindert das Erstellen von Arbeitskalender-Datumszeilen.

Bald verfügbar

Eine neue Common Data Service Lösung wird in Kürze mit den folgenden Änderungen verfügbar sein:

Beschreibung Änderung
Stelle/Position Entitätsänderungen
  • Kompensationsregion hinzugefügt
  • Finanzdimensionen hinzugefügt
Arbeitskraft Entitätsänderungen
  • Namensfolge hinzugefügt
  • Arbeitet von zu Hause hinzugefügt
  • Sprache hinzugefügt
  • Dienstalterdatum hinzugefügt
  • Jahrestag hinzugefügt
  • Ursprüngliches Einstellungsdatum hinzugefügt
Beschäftigung Entitätsänderungen
  • Finanzdimensionen hinzugefügt
  • Kündigungsgrund hinzugefügt
  • Kündigungsdatum umbenannt von Übergangsdatum
  • Probezeitdatum hinzugefügt
Arbeitskraftadresse Entitätsänderungen
  • Straße hinzugefügt
  • Adresszeile 1, Adresszeile 2, und Adresszeile 3 als veraltet markiert
Neue variable Vergütung-Einrichtungsentitäten
  • Plantyp für variable Vergütung
  • Plan für variable Vergütung
  • Übertragungsregeln
  • Variable Planstufe zur Vergütung
Neue Arbeitskraftkalender-Beschäftigung Entität
  • Arbeitkraftkalender-Entität hinzugefügt
