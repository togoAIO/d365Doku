Zuordnen von Lagerartikeln zu Serviceobjekten

Important

Dynamics 365 for Finance and Operations hat sich zu speziell entwickelten Anwendungen entwickelt, mit denen Sie bestimmte Geschäftsfunktionen verwalten können. Weitere Informationen zu diesen Änderungen finden Sie im Dynamics 365-Lizenzierungshandbuch.

In diesem Thema wird erläutert, wie Sie einen Lagerartikel an ein Serviceobjekt anhängen. Wenn Sie einen Artikel an ein Serviceobjekt anhängen, können Sie die Serviceaktivitäten steuern und melden, die für den Artikel ausgeführt werden.

Damit Serviceobjekten Artikel zugeordnet werden können, müssen die Artikel zunächst im Formular Details für freigegebene Produkte erstellt werden.

Verwenden Sie die folgenden Schritte, um einen Lagerartikel an ein Serviceobjekt anzuhängen:

  1. Klicken Sie auf Serviceverwaltung > Einstellungen > Serviceobjekte > Serviceobjekte.

  2. Wählen Sie im Feld Artikelnummer den Artikel aus, den Sie dem Serviceobjekt hinzufügen möchten.

  3. Speichern Sie das Serviceobjekt.

Der Artikel ist nun dem Serviceobjekt zugeordnet. Jede für den Artikel angegebene Lagerungsdimension wird auch auf das Serviceobjekt übertragen.
